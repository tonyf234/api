# API
download npm packages:
`npm install`

copy the '.env_sample' file to '.env' before running any script
`cp .env_sample .env`

run development:
`npm run start:dev`

run test:
`npm run start:test`

## <u>Endpoints</u> :
- /users
    - / : get all users
- /auth
  - /signup
  - /signin

## <u>workflow</u> :
1. entity
2. (repository)
3. service
4. controller
5. router
6. app
